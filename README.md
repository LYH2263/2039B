# 极简 PHP 论坛 (前后端分离版)

## 🛠 技术栈
- Frontend: Vite + Vanilla JS + Bootstrap 5.3 (NPM managed)
- Backend: Native PHP 8.2
- Database: MySQL 5.7

## 🚀 启动指南 (How to Run)
1. 确保 Docker Desktop 已启动。
2. 在根目录执行：`docker compose up --build`
3. 等待容器启动完成...
